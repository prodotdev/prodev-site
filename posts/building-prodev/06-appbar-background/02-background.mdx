# Background

To add the colored background from the design, we'll add `background` styles to our `Home` component.

**src/app/_components/Home.module.css**

```css
.root {
  height: 100vh;
  background-repeat: no-repeat;
  background: url(/home-gradient-bg.jpg) no-repeat, linear-gradient(180deg, rgba(0,0,0,0) 0%, #000 100%);
  background-size: 100%;
  background-blend-mode: overlay, normal;
}
```

- Background image saved in **public/home-gradient-bg.jpg**

Apply the styles to the home page:

**src/app/page.tsx**

```tsx
...
import styles from '@/app/_components/Home.module.css'

export default function Home() {
  return (
    <div className={styles.root}>
      ...
    </div>
  )
}
```
